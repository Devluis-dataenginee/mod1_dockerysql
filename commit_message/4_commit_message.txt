Este commit introduce la funcionalidad para cargar datos de viajes de taxi en una base de datos PostgreSQL. Se incluye el script `cargar_green_taxi.py` que maneja la ingesta de datos desde un archivo Parquet.

Se añade también la configuración de `docker-compose.yaml` para orquestar los servicios de la aplicación y la base de datos.
